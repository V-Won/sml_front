From Specimen Collection to Test Results

01.

Specimen Collection

  • Use the appropriate container indicated in the Test Request Form for each test items.
  • Please verify the collection method suitable for the test items and submit a quantity of specimen sufficient to perform the test.
Test Directory

02.

Completion of Test Request Form

  • Please select and fill out the appropriate Test Request Form according to the test item.
Test Request Form

03.

Specimen Transport

  • Dedicated staffs assigned to each client personally visit the site to collect specimens.
  • Specimens are transported safely and rapidly via specialized logistics transportation services.
Our Networks

04.

Conducting Test

  • Each requested test item is accurately analyzed in the respective department, strictly following quality management guidelines.

05.

Test Report

  • Results can be checked in real-time swiftly via Web/Mobile access available 24 hours.
  • Test result sheets are being delivered through the dedicated staff at each branch.
  •  If additional testing is required, test items can be added through the branch staff or customer center (1661-5117).
